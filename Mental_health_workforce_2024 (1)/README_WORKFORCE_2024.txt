Please read this README.txt file before using the workforce *.csv file.
 
This README.txt accompanies the workforce_2024.csv for the AIHW Meant health Workforce article. 

The files are intended for loading into analytical software for further analysis. It can be imported into a number of statistical software packages including SAS, R, or read by text editors including Notepad++. 

Tips and warnings when analysing the *.csv files:
 - Any field with a 'Number of Practitioners' of 3 or less, and any numbers derived from this, have not been published due to confidentiality and will appear as 'Null'. 
 - Null fields may also represent instances where data is unavailable, not relevant or does not exist.
 - All data have been rounded to the nearst whole number.
